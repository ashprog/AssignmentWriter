# AssignmentWriter
Download the app from playstore
https://play.google.com/store/apps/details?id=in.ashprog.assignmentwriter

Writer. is a tool using which you can write assignments in your own handwriting.

Writer. includes default handwriting font so that you don't have to worry about creating another. Just install the app and you're good to go.
You can also add your handwriting as font. Just follow the steps given in the app.

It has following features:
1) Change page's background.
2) Change font size/color/line space.
3) Download the page in PNG format.
4) Create pdf file from the downloaded pages.

Writer. uses following tool/library:
1) Calligraphr : A free tool to convert your handwriting as font.
2) iText : A library to convert images to pdf file.

Please note that Writer itself doesn't have any tool to convert your handwriting to font. We use third-party tool (Calligraphr) for the same.
